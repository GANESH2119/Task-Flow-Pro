# 🚀 TaskFlow Pro

Professional Python-based Task Management Application with Rich UI and JSON Storage.

---

## ✨ Features

✅ Add Tasks  
✅ View Tasks  
✅ Complete Tasks  
✅ Delete Tasks  
✅ Task Analytics  
✅ Rich Console UI  
✅ JSON File Storage  
✅ Priority Levels  
✅ Due Dates  

---

## 🛠 Technologies Used

- Python
- Rich Library
- JSON

---

## ▶ Run Project

Install Rich Library:

```bash
pip install rich
```

Run Application:

```bash
python main.py
```
